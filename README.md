# Crazy YAKAYAKA
Version 1.0
試作版（とりあえず動く）


## Summary
ツムが増えます。   
ツムが消えます。   
ちょっとした軽率で生まれたアプリケーション。   

## How To
ツムが増えます。   
```
$ python crazy_yakayaka.py   
```

ツムが消えます。   
```
$ python clean_yakayaka.py   
```

PIDを調べて各々killしてください。   
```
$ ps aux | grep python   
$ kill -9 XXXX   
```

## Support
- Linux
- Python 2.7.12
